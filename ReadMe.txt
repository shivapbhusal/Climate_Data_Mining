Steps : 

1. Copy the Whole Folder and keep it inside following directory in Windows : 
C:\xampp\htdocs
Or any other directory in the Linux System. Make Sure you have installed xampp or wampp in your machine. 

2. Open Xampp or Wampp and start Apache and Mysql 

3. Go to your Browser and Open PhpMyAdmin : 
http://localhost/phpmyadmin/

4. Create a database called "climate". 

5. Import climate.sql file in that database. 
That file is inside the folder called "database". 

6. Open the link : http://localhost/climate_mining/map/index.php 
and Enjoy the project.
